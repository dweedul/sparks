FONTLOG for the Sparks fonts

This file provides detailed information on the Sparks Font Software.
This information should be distributed along with the Sparks fonts and any derivative works.

Basic Font Information

Sparks is a font family for creating sparklines charts in text via OpenType contextual substitutions.

Sparks currently features three styles (bar, dot, and dot-line), each of which has five weights.

To contribute to the project contact Mike Gallagher at mike@aftertheflood.com
